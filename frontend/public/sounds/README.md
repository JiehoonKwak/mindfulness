# Bell Sound Files

Place the following audio files in this directory:

- `start-bell.mp3` - Played when meditation session starts
- `end-bell.mp3` - Played when meditation session completes

## Recommended Sources

Free, high-quality meditation bell sounds can be found at:
- [Freesound.org](https://freesound.org) - Search for "meditation bell" or "singing bowl"
- [Pixabay](https://pixabay.com/sound-effects/) - Royalty-free sound effects

## Requirements

- Format: MP3 (for best browser compatibility)
- Duration: 2-5 seconds recommended
- Volume: Normalized, not too loud
